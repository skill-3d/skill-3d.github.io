Motivation

Tool availability is not enough.

Uniform Tool Use

Existing agents often apply fixed tool workflows across different spatial tasks, causing mismatched evidence and limited gains.

Scene-Aware Skills

Skill-3D retrieves task-specific skills to select the right tools and ground answers in relevant 3D evidence.

Method

Scene Memory, Skill Library, and skill-guided post-training.

Scene-Aware Skill Extraction

Successful rollouts are distilled into reusable workflows. Failed rollouts are attached as lessons and converted into fallback rules when they provide reliable corrections.

Skill-Guided Inference

Given a query, Skill-3D retrieves candidate static and dynamic skills, selects a compact subset, and uses them to guide evidence acquisition and final reasoning.

Skill-Guided Agentic Post-Training

Skill-guided trajectories supervise compact MLLM agents through agentic SFT and GRPO, teaching them to select skills, call tools, and ground answers in evidence.

Results

Consistent gains across closed- and open-source agents.

Comprehensive Evaluation

VSI-Bench, BLINK, CV-3D, and MMSI-Bench

Closed-source evaluation across GPT-4o, GPT-5.4, Gemini-2.5-Pro, and Gemini-3-Flash.

Closed-source agents

Skill-3D consistently outperforms non-agentic, direct tool-use, and Think3D baselines across four closed-source MLLM agents. The gains are most pronounced on VSI-Bench, where different spatial categories require task-specific evidence such as object grounding, depth estimation, and multi-view verification.

Open-source agents

Skill-guided post-training transfers to compact Qwen3-VL-4B/8B agents. The results show that scene-aware tool-use behavior can be internalized by smaller policies through agentic SFT and GRPO, improving skill selection, tool usage, and evidence-grounded answering.

Analysis

Skill-3D improves tool-use quality, not just tool-call frequency.

Qualitative Cases

Dialogue examples

Measuring from the closest point of each object, what is the distance between the table and the bathtub?
#dy_06 · depth-distance#st_01 · object grounding
Pi3→Obj. Det.→Depth Est.
Skill-3D retrieves a depth-distance skill and combines reconstruction, object localization, and dense depth cues. The final answer is 0.9m.
How many chair(s) are there in this room?
#dy_12 · detection-counting#st_04 · multi-view dedup
Pi3→Obj. Det.
Skill-3D uses layout consistency and object detection to suppress duplicated chair appearances across views. The final count is 4.
What is the relative direction of the chair with respect to the table?
#dy_09 · orientation-relation#st_03 · spatial grounding
Obj. Det.→OrientAnything→Pi3
Skill-3D retrieves an orientation-relation skill and grounds the answer using object positions and orientation evidence.
Estimate the room size using the sampled views.
#dy_08 · room-size#st_02 · geometry check
Pi3→Depth Est.→Layout Check
The retrieved skill directs the agent to combine room-level reconstruction with metric depth cues before producing the final estimate.
Which object appears first along the camera trajectory?
#dy_14 · appearance-order#st_06 · multi-view temporal cue
View Sampling→Obj. Det.→Order Check
Skill-3D uses multi-view ordering evidence instead of a single-frame guess, reducing ambiguity in appearance-order reasoning.
Which route should I take to move from the door to the sofa without crossing obstacles?
#dy_15 · route-planning#st_05 · layout grounding
Pi3→Seg.→Layout Check
Skill-3D combines room-level geometry and obstacle evidence to produce a route that avoids blocked regions and remains consistent across views.
